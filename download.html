what3words for Google Maps - Extension Files

Download all the files below and maintain the same directory structure when saving them on your computer.

Root Directory

manifest.json - Extension manifest file extension.zip - Complete extension (all files zipped) generated-icon.png - Generated icon

/assets Directory

icon16.svg - Small icon icon48.svg - Medium icon icon128.svg - Large icon

/background Directory

background.js - Background script

/content Directory

content.js - Content script content.css - Content styles

/popup Directory

popup.html - Popup HTML popup.css - Popup styles popup.js - Popup script

Installation Instructions

  1. Download all files above and organize them in the same folder structure on your computer
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" using the toggle in the top-right corner
  4. Click "Load unpacked" button
  5. Select the folder containing the extension files
  6. The extension should now be installed and appear in your Chrome toolbar
  7. Click the extension icon, enter the what3words API key Y2DE0YWC, and click "Save"
  8. Navigate to Google Maps and click anywhere on the map to see the what3words address
